!!!OBS!!!
Koden förväntas köras på Solaris x86_64
Koden kan ej köras på övriga datorer pga brister i traverseStack.
!!!OBS!!!


Vår kod kompileras enklast genom att använda vår makefil som hittas i samma map som koden.

Skriv "make" för att kompilera

Skriv "make test" för att kompilera och köra testerna.

Skriv "make final" för att skapa en objektfil av våran kod.
